    BaseControl

    Direct Subclass:

    AndGate, BufferGate, Bulb, HighConstant, LowConstant, NandGate, NorGate, NotGate, OrGate, PushButton, SwitchButton, XnorGate, XorGate

    BaseControl is the super-class of all logic controls. The BaseControl class provides basic functionality that all controls should have:

    • drag-drop
    • focus
    • manage the input/output pins

    All classes that extends BaseControl should override the initControl() method.
